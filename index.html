Media Gallery

How It Works

  1. Click the "Choose Files" button to select multiple images or videos from your device
  2. Files will be displayed in a gallery view - click on any item to view it larger
  3. For images, use the crop button to crop the image as needed
  4. For videos, use the trim button to trim the video duration
  5. Your files remain completely private - they are never saved to any server
  6. Use the "Clear All" button to remove all files and start over
